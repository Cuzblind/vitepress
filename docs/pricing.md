# 定价方案

这里放你的实际倍率、套餐和活动规则。先用占位内容搭结构，后续直接改表格即可。

## 套餐示例

| 套餐 | 说明 | 适合人群 |
| --- | --- | --- |
| 按量计费 | 充值后按调用消耗额度 | 个人开发者 |
| 包月套餐 | 统一月费，限定额度或速率 | 高频用户 |
| 企业方案 | 定制倍率、白名单、工单支持 | 团队与商用 |

## 倍率示例

| 模型 | 计费方式 | 备注 |
| --- | --- | --- |
| Claude 系列 | 按输入 / 输出 Token | 写清倍率和上游映射 |
| GPT 系列 | 按模型单独计费 | 标明是否支持视觉 |
| Gemini 系列 | 按请求类型区分 | 标明免费 / 收费差异 |

## 额度说明

- 充值金额和美元额度的换算关系。
- 最低充值金额。
- 是否支持退款。
- 是否支持赠送额度或活动码。

::: warning
价格页是用户做决定前最常看的页面。不要只写“倍率”，要把换算关系和计费单位写清楚。
:::
