# 充值教程

这一页建议配截图。先把流程写清楚，后面再补图。

## 标准流程

1. 注册并登录控制台。
2. 进入余额或充值页面。
3. 选择充值金额。
4. 完成支付。
5. 返回控制台确认额度到账。
6. 创建或复制 API Key 开始调用。

## 建议说明的细节

- 支持哪些支付方式。
- 到账时间。
- 最低充值金额。
- 是否自动开票。
- 额度是否会过期。

## 常见异常

### 支付成功但余额未更新

- 刷新页面后重试。
- 等待 1 至 5 分钟。
- 携带订单号联系人工支持。

### 充值后调用仍提示余额不足

- 检查是否使用了正确账号下的 API Key。
- 检查是否有子账号额度限制。
- 检查模型倍率是否高于预期。
