# 站点介绍

OpenToken 是一个面向开发者与普通用户的 API 中转文档站，用来说明模型支持范围、调用方式、额度规则与常见问题。

## 适合放在这里的内容

- 支持哪些模型，例如 Claude、GPT、Gemini。
- 支持哪些接口格式，例如 OpenAI 兼容格式。
- 使用限制，例如速率限制、余额要求、禁用场景。
- 服务承诺，例如工单响应时间、维护公告入口。

## 建议你补充的真实信息

### 模型支持

- Claude 4 / Claude 3.7 系列
- GPT-4.1 / GPT-4o / o3 / o4-mini
- Gemini 2.5 系列

### 使用边界

- 本站仅提供 API 转发与额度管理。
- 不承诺所有上游模型永久可用。
- 如遇上游限流，响应速度和成功率可能波动。

## 推荐用户路径

1. 先看定价和支持模型。
2. 完成注册、充值和 Key 获取。
3. 按 API 示例接入。
4. 出错时优先查看 FAQ。
