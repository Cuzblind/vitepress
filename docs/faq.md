# 常见问题

## 为什么返回 401？

- API Key 无效。
- Key 已被删除或禁用。
- 请求头没有正确带上 `Authorization: Bearer ...`。

## 为什么返回 429？

- 当前账号超过速率限制。
- 上游模型正在限流。
- 余额不足也可能在部分实现中表现为请求被拒绝。

## 为什么模型提示不存在？

- 模型名与平台定义不一致。
- 当前套餐未开放该模型。
- 上游模型已下线或临时不可用。

## 为什么响应很慢？

- 上游拥堵。
- 使用了高延迟推理模型。
- 开启流式输出后首包更快，但完整耗时仍取决于模型。

## 怎么排查问题最快？

1. 先确认余额、Key、模型名。
2. 再看请求体是否符合 OpenAI 兼容格式。
3. 最后对照错误信息联系支持，并提供请求时间和模型名。
